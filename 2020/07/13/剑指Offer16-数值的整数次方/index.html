解析：

首先考虑边界问题，底为 0 时，0 的 0 次方没有任何意义，返回特定的值。

其次，

  • pow(a, n) = pow(a, n / 2) * pow(a, n / 2)（n 为偶数）
  • pow(a, n) = pow(a, n-1 / 2) _ pow(a, n-1 / 2)_ a（n 为奇数）
